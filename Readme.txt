2- Now open Eclipse IDE.
3- Click File->Open Project From File Sytems->(in import source, select the path to the "Code" folder and inside folder select "NaikiSDA" folder from "eclipse-workspace") and click Finish.
4- Now follow this path:
NaikiSDA->src->main->webapp->WEB-INF->Login.jsp
5- Right click on "Login.jsp" and click "Run As" and later click "Run on server". 
6- Select Apache Tomcat v9.0 run environment.
7- Configuration path is "/Servers/Tomcat v9.0 Server at localhost-config"
